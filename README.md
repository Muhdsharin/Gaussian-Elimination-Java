# Gaussian-Elimination-Java

## Overview
This program implements the Gaussian Elimination algorithm to solve systems of linear equations.

## Features
- Accepts user input for number of equations and variables
- Processes augmented matrix
- Displays step-by-step elimination
- Outputs final solution

## Technologies Used
- Java
- Console-based input/output

## How to Run
1. Compile the program:
   javac GaussianElimination.java

2. Run:
   java GaussianElimination

## Concept Used
- Gaussian Elimination (Linear Algebra)
- Matrix manipulation
